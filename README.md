# authcraft/cimd-dotnet

[![CI](https://github.com/authcraft/cimd-dotnet/actions/workflows/ci.yml/badge.svg)](https://github.com/authcraft/cimd-dotnet/actions/workflows/ci.yml)

AuthCraft.Cimd is a .NET library that implements the **OAuth Client ID Metadata Document (CIMD)** protocol, enabling authorization servers to dynamically discover OAuth clients from standardized metadata document endpoints.

It provides HTTP retrieval of client metadata documents plus schema and content validation, and includes a companion integration package (`AuthCraft.Cimd.DuendeIdentityServer`) for seamless use with Duende IdentityServer.
